# Robotics School Platform (Activity 7)

## Project Description
A small platform designed for a robotics school to help teachers manage their classes. The system allows users to have specific roles (Students, Teachers, and Administrative). The platform organizes students into groups (beginner, intermediate, advanced) and assigns them courses. Each course contains a title, cover, content, and is linked to a specific didactic robotics kit.
